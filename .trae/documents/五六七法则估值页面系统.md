# 五六七法则估值页面系统设计

## 功能需求
1. **输入字段**：
   - 未来6年平均ROE
   - 未来6年平均负债率DR
   - 最新的市净率PB

2. **计算逻辑**：
   - 根据负债率DR映射风险基数：
     - 低负债（低于50%）：5%
     - 中等负债（50%-70%）：6%
     - 高负债（高于70%）：7%
   - 根据负债率DR映射风险补充：
     - 低负债（低于50%）：0.7
     - 中等负债（50%-70%）：0.6
     - 高负债（高于70%）：0.5
   - 计算目标基数PB：ROE / 风险基数
   - 计算买入点BUY：目标基数PB * 风险补充（打几折，乘以折扣率）
   - 计算卖出点SAL：BUY的2倍

3. **页面展示**：
   - 输入表单
   - 计算结果展示
   - 响应式设计

## 技术实现
- 使用纯HTML、CSS和JavaScript实现
- 单页应用，无需后端服务
- 美观的现代化设计
- 实时计算和结果展示

## 关键计算函数
```javascript
// 计算风险基数
function calculateRiskBase(dr) {
  if (dr < 50) return 5;
  if (dr <= 70) return 6;
  return 7;
}

// 计算风险补充
function calculateRiskSupplement(dr) {
  if (dr < 50) return 0.7;
  if (dr <= 70) return 0.6;
  return 0.5;
}

// 计算目标基数PB
function calculateTargetPB(roe, riskBase) {
  return roe / riskBase;
}

// 计算买入点
function calculateBuyPoint(targetPB, riskSupplement) {
  return targetPB * riskSupplement;
}

// 计算卖出点
function calculateSellPoint(buyPoint) {
  return buyPoint * 2;
}
```
